Python Comments
(by m1st)

What does Python comment do?

+ Explain Python code

+ Make the code more readable

+ Prevent execution when testing code

Singular Python Comment

* Putting a '#' at the beginning of a line, Python will take the rest as a line of text

Example:

#This is a comment

* A comment can be put anywhere in a code.

When executing the code, Python will automatically skip the comments.

Example:

print("Hello World in Python!") #This is a comment!

* The code will only execute the "print" line and skip the comment.

Multi-line Python Comment

Essentially, putting many comments in the code is also fine.

#this is the first comment
#printing "Hello World" to the screen
print("Hello World")
#code finished
#this is the last comment of the code

 Another type of multi-line comment is using ' ' '

Example:

'''
This is a mutli-line comment
Prob the best comment ever lol

'''
print("Hello World!")
